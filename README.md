# Тестовое задание: CrptApi

## Описание

Это тестовое задание включает в себя реализацию класса `CrptApi` для работы с API Честного знака.
Класс должен быть потокобезопасным и поддерживать ограничение на количество запросов к API.
Ограничение указывается в конструкторе в виде количества запросов в определенный интервал времени.

## Функциональность

Класс `CrptApi` предоставляет единственный метод `createDocument`, который отправляет HTTP POST
запрос для создания документа в API Честного знака. Документ и подпись передаются в метод в виде
Java объекта и строки соответственно.

## Используемые библиотеки

- Apache HttpClient для отправки HTTP запросов.
- Jackson для сериализации объектов в JSON.
- JUnit 5 для написания тестов.
- Mockito для создания mock объектов.
